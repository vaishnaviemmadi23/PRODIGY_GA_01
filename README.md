# GPT-2 Text Generation Project

This project demonstrates text generation using the GPT-2 model from Hugging Face Transformers.

## Objective
To generate human-like text based on user input prompts using a pre-trained GPT-2 model.

## Tools Used
- Python
- Google Colab
- Hugging Face Transformers
- GPT-2 Model

## How it works
The user enters a prompt, and the GPT-2 model generates meaningful text based on the input.

Artificial Intelligence is

## Example Output
Artificial Intelligence is transforming the world by improving automation and decision-making.
